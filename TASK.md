# Available Features

You can now use the following features in the GUI:

- 📁 **Open Repository** - Completed
- 👀 **View Status** - Check file modification status
- 📝 **Commit Changes** - Create new commits
- 🌿 **Manage Branches** - View, create, and switch branches
- 📜 **View History** - Review commit logs and history
- 🔄 **Refresh** - Update all views
